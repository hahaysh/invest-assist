# Google Sheets Schema

## 스프레드시트 구성

탭 2개:
1. `일일브리핑`
2. `주간포트폴리오리포트`

> 시트에는 숫자·짧은 메모만 저장. 긴 본문은 markdown 파일 경로로 대체.

## 데이터 입력 규칙
- 숫자형 컬럼: 단위 기호 제외 (%, $, ₩ 표기 금지), 순수 숫자만
- 텍스트형 컬럼: 50자 이내 권장, 초과 시 핵심만 축약
- 리포트파일 컬럼: markdown 파일의 상대 경로만 기록 (본문 내용 금지)
- 날짜/시간: 한국 시간(KST) 기준
