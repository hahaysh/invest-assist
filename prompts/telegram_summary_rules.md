# Telegram Summary Rules

## 전송 조건
- **Google Sheets 기록 성공 후에만 전송한다.**
- Google Sheets 실패 시: 전송 스킵 + 로그에 "Telegram skipped: Sheets write failed" 기록.

---

## Daily 요약 (5~8줄)

### 형식
```
📅 {YYYY-MM-DD} 일일 투자 브리핑

🌍 시장: {코스피 등락} | {S&P500 등락} | USDKRW {환율}
📌 오늘 볼 것: {핵심 시장 동인 1~2줄}
💼 포트폴리오: {중요 변화 또는 "변화 없음"}
⚠️ 즉시 확인 리스크: {리스크 1~2줄 또는 "없음"}
📋 주요 일정: {당일 주요 이벤트}
📁 상세: reports/daily/{YYYY-MM-DD}.md
```

### 규칙
- 총 줄 수: 5~8줄 이내
- 종목명은 ticker로 표기 (예: 삼성전자→005930, SK하이닉스→000660)
- 수치는 간결하게 (예: +1.2%, -0.8%)
- "오늘 볼 것", "포트폴리오 중요 변화", "즉시 확인할 리스크" 세 가지는 반드시 포함
- 브리핑 전체 요약 금지 — 핵심만

## Weekly 요약 (6~10줄)

### 형식
```
📊 {YYYY} {WEEK_NUM} 주간 포트폴리오 리포트

📈 성과: 포트폴리오 {주간수익률} vs 벤치마크 {벤치수익률} ({초과/미달})
🏆 기여 Top: {종목 ticker} ({수익률})
📉 기여 Bot: {종목 ticker} ({수익률})
💼 Thesis 변화: {변경된 종목 또는 "없음"}
🔄 리밸런싱: {후보 종목 또는 "검토 불필요"}
🎯 다음 주: {주요 촉매 1~2개}
⚠️ 주간 리스크: {핵심 리스크}
📁 상세: reports/weekly/{YYYY}-{WEEK_NUM}.md
```

### 규칙
- 총 줄 수: 6~10줄 이내
- 수치 필수 포함 (수익률, 초과수익률)
- Thesis 훼손 종목은 반드시 명시
- 리밸런싱 후보가 있으면 ticker와 방향 (매수/매도) 포함
- 장황한 분석 금지 — 결론과 숫자 중심
